glacier_mapping.data.mask.clip_shapefile(img_bounds, img_meta, shps)¶

Clip Shapefile Extents to Image Bounding Box

Parameters

  • img_bounds – The rectangular lat/long bounding box associated with a raster tiff.

  • img_meta – The metadata field associated with a geotiff. Expected to contain transform (coordinate system), height, and width fields.

  • shps – A list of K geopandas shapefiles, used to build the mask. Assumed to be in the same coordinate system as img_data.

Return result

The same shapefiles as shps, but with polygons that don’t overlap the img bounding box removed.

glacier_mapping.data.mask.generate_mask(img_meta, shps)¶

Generate K-Channel Label Masks over Raster Image

Parameters

  • img_meta – The metadata field associated with a geotiff. Expected to contain transform (coordinate system), height, and width fields.

  • shps – A list of K geopandas shapefiles, used to build the mask. Assumed to be in the same coordinate system as img_data.

Return mask

A K channel binary numpy array. The k^th channel gives the binary mask for the k^th input shapefile.
