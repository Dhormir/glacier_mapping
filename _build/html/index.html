Welcome to Glacier Mapping’s documentation!¶

Training/Validation Pipeline:
1- Initialize loaders (train & validation)

1.1-Pass all params onto both loaders

2- Initialize the framework 3- Train Loop args.e epochs

3.1 Pass entire data loader through epoch 3.2 Iterate over dataloader with specific batch

4- Log Epoch level train loss, test loss, metrices, image prediction each s step. 5- Save checkpoints after 5 epochs 6- -n is the required parameter (name_of_the_run) 6- models are saved in path/models/name_of_the_run 7- tensorboard is saved in path/runs/name_of_the_run
